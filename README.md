# AI Agent Project

Dieses Projekt ist ein Testprojekt zur Veranschaulichung des Agentic Frameworks **Agno**. Es demonstriert, wie man mit Python und Agno intelligente Agenten erstellen kann, die mit KI-Modellen wie OpenAI's GPT-4o interagieren.

Das Projekt zeigt die grundlegenden Konzepte und die praktische Anwendung des Agentic Frameworks für die Entwicklung von AI-gestützten Anwendungen mit Python.
